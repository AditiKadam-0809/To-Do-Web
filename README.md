# To-Do List Web Application

## 📌 Project Description
This project is a simple To-Do List web application that allows users to manage daily tasks efficiently. Users can add and delete tasks dynamically.

## 🚀 Features
- Add new tasks
- Delete tasks
- Simple and user-friendly interface

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📁 Project Structure
- index.html → Main To-Do List page  
- style.css → Styling of the application  
- script.js → Functionality (add/delete tasks)  
- about.html → Project information  
- contact.html → Contact details  
- project.html → Other projects
